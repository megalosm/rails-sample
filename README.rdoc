== Install

1. Install Ruby, Rails and MySQL

2. $ bundle install

3. $ rake db:setup (for run db:create, db:schema:load, db:seed)

4. $ bundle exec cucumber (for run automated testing)

5. Test app (See username and password in seed file)


== Technologies and Frameworks

* Rails 4.1.5

* MySQL

* jQuery and CoffeeScript

* Cucumber (Use faker and factory_girl for generate data)

* Devise

* Capistrano

* Responsive with Flexbox (http://css-tricks.com/snippets/css/a-guide-to-flexbox/)
